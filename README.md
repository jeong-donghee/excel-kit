# excel-kit

[![License: MIT](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Java](https://img.shields.io/badge/Java-21%2B-orange.svg)](https://adoptium.net/)

Apache POI 기반 엑셀 생성을 더 간결하게 만드는 자바 라이브러리.
목록 익스포트 · 표/차트/이미지 리포트 · 템플릿 리포트를 관심사별 모듈로 제공한다.

## 빌드

```bash
./mvnw clean verify
```

Maven Wrapper 포함 (JDK 21+ 필요). 커밋은 [Conventional Commits](https://www.conventionalcommits.org/),
버전은 [SemVer](https://semver.org/)를 따른다.
